STATEMENT OF WORK #SOW-2025-0042

HIGH-RISK AUTOMATION IMPLEMENTATION PROJECT

This Statement of Work ("SOW") is entered into as of March 1, 2025, pursuant to the Master Service Agreement dated January 15, 2025 ("MSA") between TechCorp Systems, Inc. ("Customer") and Acme Integrations LLC ("Service Provider").

1. PROJECT OVERVIEW

1.1 Project Name: Enterprise Automation Platform Implementation
1.2 Project Duration: 18 months
1.3 Project Value: $2,500,000.00

2. SCOPE OF WORK

Service Provider shall implement a comprehensive industrial automation platform including:

2.1 Phase 1 - Discovery & Design (Months 1-3)
    - Current state assessment
    - Requirements gathering
    - System architecture design
    - Integration specifications

2.2 Phase 2 - Development & Integration (Months 4-12)
    - Custom software development
    - Hardware integration
    - Third-party system connections
    - Database migration

2.3 Phase 3 - Testing & Deployment (Months 13-18)
    - User acceptance testing
    - Performance validation
    - Production deployment
    - Training and documentation

3. DELIVERABLES

3.1 Service Provider shall deliver:
    - System design documents
    - Custom software modules
    - Integration connectors
    - Test scripts and results
    - User documentation
    - Training materials
    - As-built documentation
